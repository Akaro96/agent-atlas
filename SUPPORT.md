# Support

## What This Project Supports

This repository supports:

- the portable templates
- the install flow
- the validation flow
- the sample workspace and sample vault

## What This Project Does Not Support

- debugging your private production environments
- repairing external agent vendor outages
- migrating every custom local script from an unrelated personal setup

## Before Opening an Issue

1. Run `pwsh -File .\scripts\Validate-AgentWorkspaceKit.ps1`
2. Check whether the problem is in the generated workspace or only in your personal customizations
3. Include the smallest reproducible example you can

# Agent Atlas Logo Assets

These files are for **project branding**, not for a maintainer's personal GitHub profile image.

## Current files

- `agent-atlas-mark-square.png`: current master raster export with transparent background
- `agent-atlas-mark-avatar-512.png`: compact square version for README, repo-facing previews, and small placements
- `agent-atlas-mark-256.png`: lightweight README-sized variant
- `agent-atlas-mark-128.png`: tiny fallback variant for small previews

## Usage guidance

- Keep a maintainer's GitHub account avatar personal and unchanged.
- Use the project mark in the repository README, social preview workflow, release notes, screenshots, and docs where project identity matters.
- Prefer `agent-atlas-mark-256.png` for inline README display so the page stays lightweight.
- Prefer `agent-atlas-mark-avatar-512.png` when testing small-size legibility.

## Quality note

The current mark is a cleaned transparent PNG derived from a raster source.
It is good enough for repository branding, but a future SVG redraw would still be the strongest long-term master asset.

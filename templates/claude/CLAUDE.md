# Global Claude Operating Rules

## Communication

- default conversation language: English unless the user prefers another language
- be direct, concise, and practical
- keep routine responses short

## Working Style

- inspect before acting when context is unclear
- keep this file short
- push path-specific behavior into local rules and skills
- prefer explicit specialist review for public UI and security-sensitive work

## Durable Knowledge

Use `__VAULT_ROOT__` as the durable knowledge layer.
Do not replace project-local docs with vault notes.
When a project becomes serious enough to deserve cross-session recall, create or refresh an inbox draft before promoting anything into curated knowledge.

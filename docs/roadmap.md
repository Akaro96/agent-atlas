# Roadmap

## Near Term

- richer cross-platform docs
- sample repo with more realistic architecture docs
- optional Obsidian terminal profile templates
- optional Context7 integration docs
- note-lint and curation scoring for the knowledge pipeline

## Mid Term

- richer Windows and macOS installers
- portable public-web audit starter module
- machine-readable manifest for optional modules

## Non-Goals

- turning the kit into a giant workflow operating system
- shipping personal memory or private project history
- pretending one harness shape fits every user equally well

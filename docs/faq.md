# FAQ

## Do I need both Codex and Claude Code?

No.
The kit is designed so you can adopt one agent runtime first and add the second later.

## Do I need Obsidian?

No.
The vault is optional.
It exists to keep durable notes separate from operational project folders.

## Does the installer overwrite my real Codex or Claude Code config?

No.
It writes portable starter configs into the generated destination root.
Review and merge those files into your live setup deliberately.

## Is this safe by default?

Safer than many ad-hoc setups, yes.
But it is not a locked-down security product.
The templates intentionally expose a balanced and understandable starting point, not maximum friction.

## Can I use only the workspace part?

Yes.
That is a valid way to start.

## What should I customize first?

1. the Codex trust roots
2. the Claude default permission mode
3. the workspace and vault names
4. your preferred specialist or review rules
